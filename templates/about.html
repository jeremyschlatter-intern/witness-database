About This Database

What Is This?

The Congressional Witness Database is a searchable index of witnesses who have testified before House and Senate committees from the 116th Congress (2019) to the present. It lets you look up who testified, when, before which committees, and in what capacity.

This tool is designed for congressional staff, journalists, researchers, and anyone who needs to quickly answer questions like:

  • Who has served as FAA Administrator and when did they testify?
  • How many times has a particular person testified before Congress?
  • Which witnesses appeared before the House Judiciary Committee this year?
  • What hearings covered a specific topic?

Data Sources

All data comes from official government APIs:

  • Congress.gov Committee Meeting API — Structured witness data including names, titles, organizations, and links to written statements. This provides comprehensive witness data for House hearings from the 116th Congress onward.
  • Congress.gov Hearing API — Hearing metadata and links to GovInfo transcripts.
  • GovInfo API — Full hearing transcripts in HTML and PDF formats.

Known Limitations

  • Senate witness data: The Congress.gov Committee Meeting API does not provide structured witness information for most Senate hearings. Senate hearings are listed, but witnesses may not be indexed unless the API provides them.
  • 117th Congress House data: The Congress.gov API currently returns an error for 117th Congress House committee meetings, so this data is unavailable.
  • Witness deduplication: The same person may appear under slightly different name variants. We use conservative matching to merge obvious duplicates (e.g., "Steve" and "Stephen" Dickson), but some edge cases may remain.
  • Historical coverage: The database currently covers the 116th through 119th Congresses (January 2019 to present). Earlier hearings are not yet included.

Features

  • Smart search: Search by witness name, hearing topic, title/position, organization, or agency abbreviation (e.g., FAA, FBI, EPA, SEC).
  • Title tracking: See who has held a given title (e.g., "Administrator") and when they testified in that capacity.
  • CSV export: Download witness data as CSV for your own analysis.
  • Direct links: Every hearing links to the original transcript on GovInfo and the meeting record on Congress.gov.
  • Unique witness IDs: Each witness has a stable ID (e.g., W-00001) for reliable cross-referencing.

API Access

This database provides a JSON API for programmatic access:

  • /api/witnesses — List witnesses (supports ?q= search and ?limit=)
  • /api/stats — Database statistics
